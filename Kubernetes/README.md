# Kubernetes 特性

      kubernetes优势:

     （1）可移动：公有云、私有云、混合云、多态云
     （2）可扩展：模块化、插件化、可挂载、可组合
     （3）自修复：自动部署、自动重启、自动复制、自动伸缩
